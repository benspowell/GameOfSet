General Rules

  • The objective of this game is to identify a set of 3 cards from the 12 cards place faced up on the screen.
  • Each card has four features, which can vary according to its shape, color, number and shading.
  • A set consists of 3 cards in which each of the cards' features, looked at one-by-one, are the same on each card, or are different on each card. All the features must separately satisfy this rule.
  • Players would be awarded a point if the cards they selected are a set.

Multiplayer Mode

  • For multiplayer mode, users are required to enter the number of players who wants to participate.
  • The LIMIT for the number of players in the multpiplayer mode is 10 players
  • Each player is allowed to choose their own id/nickname prior to the start of the game.
  • During gameplay, the player who has found a set is required to click their own id button before selecting the cards.
  • The above step is required each time before selecting any sets in the game. This is to ensure the smooth gameplay.

BEWARE!

Failing to click the player id buttons prior to selecting a set in the Multiplayer Mode would not award points to the correct players. Hence, game developers would not be hold accountable to any points, respect or friendship loss after the game.
